
# Portfolio

Simple CV single-page-application project created with vite react-ts.

https://karolhas.github.io/cv/

## Run Locally

Clone the project

```bash
  git clone https://github.com/karolhas/cv.git
```

Go to the project directory

```bash
  cd cv
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
